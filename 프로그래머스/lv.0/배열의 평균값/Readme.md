
# 배열의 평균값

1.	reduce 함수는 배열의 모든 요소를 순회하면서 누적 값을 계산합니다.
2.	a는 이전 단계에서의 누적 값(누산기)이며, b는 현재 배열 요소의 값입니다.
3.	a + b는 현재까지의 누적 합계를 계산합니다.
4.	초기값은 0으로 설정되었으므로, 첫 번째 요소는 a + b가 됩니다.
5.	이 과정을 거쳐 numbers 배열의 모든 요소가 더해져서 answer 변수에 저장됩니다.
